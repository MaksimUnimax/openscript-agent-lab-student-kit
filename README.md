# OpenScript Agent Lab Student Kit

> Важно: это не рабочий проект ученика.
>
> Это публичный источник стартовой документации: документы, правила, шаблоны и заготовки.
> Его нельзя использовать как место разработки проекта.
> Ученику и Codex нельзя менять, редактировать, пробовать править или пушить сюда в рамках работы над проектом ученика.
> Отсюда можно только взять стартовые материалы.
> Рабочий проект создаётся отдельно на сервере ученика и ведётся в отдельном git ученика.

Это общий учебный комплект OpenScript Agent Lab.

- репозиторий публичный;
- здесь нет секретов и runtime-данных;
- начинать нужно с `START_HERE.md`;
- личный проект ученика создаётся отдельно на его сервере.
