# Student project seed

Это базовая структура, которую ученик создаёт на своём сервере.

Все файлы из `student-kit` копируются в личный repo ученика.
Файл `rules/codex/AGENTS.md` должен стать `<STUDENT_PROJECT_ROOT>/AGENTS.md`.
Без `AGENTS.md` Codex не должен начинать работу в личном проекте ученика.
