# Генерация расширения ChatGPT

Эта папка описывает, как ChatGPT должен по ходу работы с учеником сгенерировать браузерное расширение для вставки prefix/reminder.

Здесь не хранится готовый zip расширения.

Готовый zip расширения ChatGPT создаёт позже в конкретном диалоге с учеником, когда это понадобится по стартовому сценарию.

Расширение нужно, чтобы ChatGPT не терял правила учебного режима при длинной работе.
Оно должно позволять:
- редактировать prefix;
- включать и выключать вставку;
- копировать prefix;
- вставлять prefix в сообщение ChatGPT;
- хранить данные локально в браузере;
- не отправлять данные наружу.

Будущая версия расширения также должна иметь две служебные кнопки:

- “Копировать prompt обновления документов”;
- “Копировать prompt старта нового диалога”.

Эти кнопки должны копировать готовые prompt’ы в буфер обмена, а не отправлять их автоматически.
