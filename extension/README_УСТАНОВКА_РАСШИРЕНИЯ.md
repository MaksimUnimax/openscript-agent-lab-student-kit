# Extension

Этот раздел не содержит готовый продукт расширения.

Здесь лежат:
- prefix/reminder для будущего расширения;
- инструкция для ChatGPT, как позже сгенерировать расширение в диалоге с учеником.

Готовый zip расширения не хранится заранее.

Когда ученик дойдёт до этапа установки расширения, ChatGPT сгенерирует zip расширения, даст файл для скачивания и отдельно объяснит ученику простыми словами, как его установить.
