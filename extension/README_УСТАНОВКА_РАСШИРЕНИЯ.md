# Extension

## Что такое расширение ChatGPT

Расширение ChatGPT - это небольшой инструмент в браузере.
Оно помогает вставлять в сообщение ChatGPT специальный prefix с правилами проекта.

Prefix - это текстовое напоминание для ChatGPT.
В нём написано, как вести ученика:
- читать правила и ТЗ;
- не гадать;
- идти одним шагом;
- проверять отчёты Codex;
- давать Codex prompt цельным блоком;
- объяснять простым русским языком.

## Зачем оно нужно

ChatGPT может терять контекст и забывать правила длинного проекта.
Из-за этого он может начать:
- сочинять;
- галлюцинировать;
- пропускать проверку документов;
- давать слишком большие шаги;
- отдавать Codex роль ведущего;
- нарушать формат prompt.

Расширение уменьшает этот риск, потому что добавляет правила прямо в сообщение ученика.

## Что можно делать в расширении

Ученик может:
- включить вставку prefix;
- выключить вставку prefix;
- отредактировать текст prefix;
- сохранить изменения;
- скопировать prefix вручную;
- проверить, что prefix вставляется в ChatGPT.

## Важно

Расширение не заменяет ChatGPT.
Расширение не заменяет Codex.
Расширение не пишет код проекта.
Расширение только помогает удерживать правила работы в каждом сообщении.

## Дополнительные кнопки prompt'ов

Будущее расширение должно иметь две дополнительные кнопки:

1. “Копировать prompt обновления документов”
2. “Копировать prompt старта нового диалога”

Эти кнопки нужны, чтобы ученик не искал длинные служебные prompt’ы вручную.

Кнопка обновления документов копирует prompt, который помогает ChatGPT сохранить текущий рабочий контекст в документацию проекта.

Кнопка старта нового диалога копирует prompt, который помогает ChatGPT начать новый диалог и восстановить работу по repo-based документации.

Важно:
- кнопки только копируют текст в буфер;
- они не отправляют prompt автоматически;
- ученик сам вставляет prompt в ChatGPT;
- prompt’ы должны использовать данные проекта ученика;
- нельзя оставлять чужие ссылки на repo, чужие SSH URL или чужие server paths.

> Этот раздел — часть публичного источника стартовой документации.
> Здесь лежат только материалы для чтения и копирования в будущий рабочий проект ученика.
> Это не место разработки и не рабочий repo ученика.
> Отсюда можно только взять prefix и инструкцию для будущей генерации расширения.

Этот раздел не содержит готовый продукт расширения.

Здесь лежат:
- prefix/reminder для будущего расширения;
- инструкция для ChatGPT, как позже сгенерировать расширение в диалоге с учеником.

Готовый zip расширения не хранится заранее.

Когда ученик дойдёт до этапа установки расширения, ChatGPT сгенерирует zip расширения, даст файл для скачивания и отдельно объяснит ученику простыми словами, как его установить.
