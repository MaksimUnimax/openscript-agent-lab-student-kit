# Prefix

## Что такое prefix

Prefix - это текст с правилами проекта.
Он вставляется перед сообщением ученика в ChatGPT.

Prefix нужен, чтобы ChatGPT каждый раз видел:
- актуальные правила работы;
- требование проверять ТЗ;
- требование не гадать;
- требование идти одним шагом;
- требование давать Codex prompt одним цельным блоком;
- требование проверять отчёты Codex.

Prefix можно редактировать.
Если проект меняется, prefix тоже можно обновлять.
Если расширение отключено, prefix можно скопировать вручную.

Важно:
prefix не должен содержать секреты, токены, пароли, приватные ключи или закрытые данные.
