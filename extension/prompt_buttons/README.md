# Prompt buttons для расширения

Эта папка описывает две дополнительные кнопки будущего расширения ChatGPT.

Расширение должно уметь не только вставлять prefix, но и копировать в буфер два готовых prompt’а:

1. Prompt обновления документов.
2. Prompt старта нового диалога.

Эти prompt’ы нужны для двух частых ситуаций:

- нужно сохранить текущий рабочий контекст проекта в документацию;
- нужно начать новый диалог и восстановить работу по repo-based документации.

Важно:

- prompt’ы должны быть доступны из интерфейса расширения отдельными кнопками;
- нажатие кнопки копирует prompt в буфер обмена;
- prompt не должен автоматически отправляться;
- ученик сам вставляет prompt в ChatGPT;
- prompt’ы не должны содержать чужие repo URL;
- все repo/path должны быть placeholders конкретного ученика;
- перед генерацией расширения ChatGPT должен заменить placeholders на данные проекта ученика или остановиться и запросить недостающие данные.

Минимальные кнопки:

- “Сохранить”
- “Вставить сейчас”
- “Копировать prompt обновления документов”
- “Копировать prompt старта нового диалога”
- “Сбросить”
