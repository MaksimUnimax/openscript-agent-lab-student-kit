# START HERE

1. Загрузите в ChatGPT два документа из `chatgpt/`.
2. Установите расширение из `extension/`.
3. Дальше ChatGPT будет вести ученика по шагам.
4. `roadmap/` и `project-docs/technical_spec/` уже лежат в repo.
5. Правила лежат в `rules/`.
6. Префикс для расширения лежит в `extension/prefix/openscript_student_prefix.md`.
7. Документы состояния ученика будут заполняться в его личном проекте.
8. ChatGPT будет напоминать об обновлении docs по счётчику.
9. Не открывайте `localhost` как адрес для ученика.
10. Личный сервер и личный проект создаются отдельно.
