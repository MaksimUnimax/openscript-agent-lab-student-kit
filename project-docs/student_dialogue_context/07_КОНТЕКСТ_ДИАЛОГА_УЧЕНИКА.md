# 07. Контекст диалога ученика

## Назначение документа

Этот документ хранит контекст диалога ChatGPT с учеником, чтобы новый диалог мог продолжить работу без потери состояния.

## Кто ученик

- имя или псевдоним:
- уровень:
- что уже умеет:

## Что уже сделано

Заполняется по фактам из отчётов и проверок.

## Где остановились

Один текущий stop-point без лишних деталей.

## Последний успешный run

- RUN_ID:
- что подтвердил:
- какой результат был принят:

## Последний проблемный run

- RUN_ID:
- что сломалось:
- какой proof ещё нужен:

## Какие документы уже загружены

- roadmap;
- ТЗ;
- карта модулей;
- актуальный снимок;
- current status;
- другой полезный контекст.

## Какие решения приняты

- что уже выбрано;
- какие правила зафиксированы;
- какие shortcuts запрещены.

## Какие ошибки нельзя повторять

- localhost как user-facing URL;
- смешивание source и runtime;
- попытка обойти Hermes;
- fake/template success;
- hardcode одного агента;
- работа по памяти вместо фактов.

## Текущие 3 задачи

1. что делаем сейчас;
2. что проверяем или закрываем;
3. что будет следующим шагом.

## Следующий шаг

Коротко и конкретно.

## История обновлений

- дата;
- что добавили;
- какой run это подтвердил.
