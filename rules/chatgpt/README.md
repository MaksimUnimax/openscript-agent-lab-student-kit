# ChatGPT rules

Папка с правилами для ChatGPT при ведении ученика.

Главный файл правил:

- `01_ПРАВИЛА_ДЛЯ_CHATGPT.md`

В нём закреплено:

- ChatGPT ведёт проект;
- Codex только исполнитель;
- ChatGPT сам читает доступные правила, ТЗ, roadmap и контекст;
- новый инструмент объясняется ученику при первом появлении;
- сначала идёт простое человеческое объяснение, потом техника;
- новый термин объясняется сразу одной короткой фразой;
- ученик может в любой момент попросить “объясни проще”;
- ChatGPT обязан остановиться и объяснить проще;
- Codex prompt даётся одним цельным блоком для копирования;
- отчёты Codex проверяются перед следующим шагом.

- `01_ПРАВИЛА_ДЛЯ_CHATGPT.md` - базовые рабочие правила.
- `02_СТИЛЬ_ОБЪЯСНЕНИЙ.md` - как объяснять простым языком.
- `03_ПРАВИЛО_БЕЗ_LOCALHOST.md` - какие адреса нельзя давать ученику.
- `04_ПРАВИЛО_ОДНОГО_ШАГА.md` - как дробить работу на один шаг за раз.
- `05_ПРАВИЛО_ПРОВЕРКИ_ОТЧЁТОВ.md` - как проверять отчёты Codex.
